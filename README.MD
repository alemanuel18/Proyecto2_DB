## Transacciones explГӯcitas

El registro de una venta (`POST /api/ventas`) usa una transacciГіn explГӯcita:

```
BEGIN
  INSERT INTO Venta ...
  FOR EACH producto:
    SELECT stock FOR UPDATE        вҶҗ bloqueo optimista
    IF stock < cantidad вҶ’ ROLLBACK + error 400
    INSERT INTO Detalle ...
    UPDATE Producto SET stock = stock - cantidad
COMMIT
```

Si cualquier paso falla (stock insuficiente, producto inexistente, error de BD), se ejecuta `ROLLBACK` y ningГәn cambio queda guardado.
